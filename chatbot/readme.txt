1- Executar o comando para instalar as dependências:
pip install -r requirements.txt

2- Inserir a chave da api do Google (api_key) no arquivo app.py no local indicado
Obter a chave na URL: https://aistudio.google.com/app/apikey

3- Substituir a URL correta para o java script acionar a URL do chat_bot
static/js/script.js 

4- Para iniciar a aplicação Flask, executar o script execute_app.sh
